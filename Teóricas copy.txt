5) Ao contrário do JavaScript, o TypeScript usa tipagem de dados. Se um dado foi 'declarado' como uma string, seu tipo não poderá ser mudado ao longo do código

O TypeScript acusa um erro de tipagem já na etapa de compilação:

JavaScript:

x = 'a'
console.log(x) -> a
x = 2
console.log(x) -> 2

TypeScript:

x = 'a'
console.log(x) -> a
x = 2 -> erro, variável é do tipo string
